Basic <img> Usage & Cumulative Layout Shift (CLS)

This page demonstrates the importance of using width and height attributes on images to prevent Layout Shift.

DevTools Experiment:
  1. Open DevTools (F12) and go to the Network tab.
  2. Set throttling to "Fast 3G" or "Slow 3G".
  3. Reload the page and observe how the text jumps when the first image loads (Layout Shift).
  4. The second image reserves its space immediately, preventing the content below from jumping.

Image 1: Without explicit dimensions (Causes Layout Shift)

Notice how this text is initially pushed down when the image finishes loading, because the browser didn't know how much space to reserve.

Some content below the image that gets pushed down...

Image 2: With width and height attributes (No Layout Shift)

The browser reserves the correct aspect ratio space before the image even loads. No text jumping!

Content below this image stays exactly where it is.
