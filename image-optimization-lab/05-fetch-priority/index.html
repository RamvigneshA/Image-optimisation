Fetch Priority

Use fetchpriority="high" to tell the browser which images are critically important (like the LCP hero image).

DevTools Experiment:
  1. Open DevTools and go to the Network tab.
  2. Filter by "Img". Right-click the column headers and ensure the "Priority" column is visible.
  3. Reload the page and observe the priority assigned to each image.
  4. The Hero image should be "High", the normal image "Low" (initially), and the lazy image "Lowest" (or unrequested until scrolled).
