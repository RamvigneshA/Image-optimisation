Format Fallbacks with <picture>

Serve next-gen image formats like AVIF or WebP to browsers that support them, while falling back to JPEG for older browsers.

DevTools Experiment:
  1. Open DevTools and go to the Network tab.
  2. Filter by "Img".
  3. Reload the page and see which format downloaded (Type column). It will likely be avif if you are on a modern browser.
  4. To see the fallback, use the "Network request blocking" tab in DevTools to block *f_avif*, then reload to see WebP load!

Next-Gen Format Fallback

The browser reads from top to bottom and downloads the first format it supports.

<picture>
  <source srcset="...avif" type="image/avif">
  <source srcset="...webp" type="image/webp">
  <img src="...jpg" alt="...">
</picture>
