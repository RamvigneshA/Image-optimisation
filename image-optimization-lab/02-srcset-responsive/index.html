Responsive Images with srcset and sizes

Serving the right image size for the user's viewport saves bandwidth.

DevTools Experiment:
  1. Open DevTools and go to the Network tab.
  2. Filter by "Img" and clear the network log.
  3. Ensure "Disable cache" is checked.
  4. Resize your browser window from mobile width (< 600px) to desktop width (> 1000px) and reload. Observe which image priority is requested and what the file size is compared to others.

Resolution Switching

The browser automatically picks the most appropriate image based on the viewport width (vw) and screen density.

400w: Loaded on small screens (< 400px wide).

800w: Loaded on medium screens.

1200w: Loaded on large screens or high-density (Retina) displays.
