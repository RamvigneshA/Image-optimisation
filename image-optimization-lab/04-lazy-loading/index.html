Native Lazy Loading with loading="lazy"

Defer loading of offscreen images until the user scrolls near them, saving initial bandwidth and CPU.

DevTools Experiment:
  1. Open DevTools and go to the Network tab.
  2. Filter by "Img".
  3. Notice how only the first (eager) image loaded initially.
  4. Scroll down the page slowly and observe new image requests triggering just before you reach them.
