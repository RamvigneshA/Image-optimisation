Custom Next.js <Image /> Engine

This page uses a custom Web Component (<smart-image>) that automatically fetches a JSON manifest and builds a complex, perfectly-optimized <picture> tag on the fly.

Scenario 1: Build-Time (Manifest System)

This image uses the image-manifest.json we generated during the build step.

Scenario 2: Dynamic (On-Demand System) 🌟

This image DOES NOT exist in the dist/ folder. It is being generated on-the-fly by our local Express server right now!

Notice how easy it is to add a new image now. No build step required for Scenario 2!
